# 2 PHP 变量

例如：

PHP 变量不同于 java C++ 等语言中的变量需要提前申明 变量类型，因为 php 会根据所附的值自动把变量转为正确的数据类型

# 2.1 php 变量作用域

  • lcoal (局部)
  • global (全局)
  • static (静态)

函数之外声明的变量拥有 GLobal 作用域，只能在函数以外进行访问

函数内部申明的变量拥有 LOCAL 作用域，只能在函数内部访问

例子：

GLOBAL 关键字

PHP 同时在不名为 $GLOBALS [index] 的数组中存储了所有的全局变量。下标为变量名，而且这个数组可在函数内部访问，并能够用于直接更新全局变量

static 关键字

静态变量的生命周期不会随着函数的执行完毕而消失，而是与整个应用的生命周期相同

# 2.2 变量命名规则

  • 变量以 $ 符号开头，其后变量的名称
  • 变量名称必须以字母或下划线开头
  • 变量名称不能以数字开头
  • 变量名称只能包含字母数字字符和下划线
  • 变量名称对大小写敏感

# 2.3 常量

如需设置常量，请使用 define() 函数 - 它使用三个参数

  1. 首个参数定义常量的名称
  2. 第二个参数定义常量的值
  3. 可选的第三个参数规定常量名是否对大小写敏感 / 默认是 false

常量是自动是全局的，而且可以贯穿整个脚本使用

# 3 php echo 和 print

echo 和 print 的差别

  • echo 能够输出一个以上的字符串
  • print 只能输出一个字符串并始终返回 1 故而要比 echo 慢
