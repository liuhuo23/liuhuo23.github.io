# 渗透主机

# 1. 确定主机

netdiscover 主机发现

扫描结果：可以根据靶机的 mvc 地址确定

# 2. 主机信息探测

信息探测： nmap -A -p- 192.168.1.48

可以发现只开放了 80，22 端口。对应的服务 为 http，ssh

网站探测

gobuster 是一个扫描网站目录的工具

挨个访问子目录，查看有无切入点

发现有 admin 等疑似登录页面的反馈成功 即 status 为 200 的项

挨个访问，寻找切入点。最终发现，只有登录页面这么一个切入点，没有其他能绕开登录的。

没办法只能爆破。

但是发现登录名是邮箱，因此我们必须想办法获取他的邮箱。

# 3. 漏洞查找

爆破登录点

首先使用 cewl 获取字典，以下两种命令都行

用户邮箱，你可以试着查看其他这个网站的其他页面，找到所有疑似邮箱的字符串然后挨个试

通过查找发现疑似 contack@hacknos.com 然后爆破

查看网络源代码 找到 表单，进一步找到 用户名和密码的参数，然后随机输入一个密码 发现网站返回 Wrong email 的提示字样。

这是一个随机生成密码的程序，因此如果破解不了，你可以试着去多运行几遍

登录进去查找切入点
