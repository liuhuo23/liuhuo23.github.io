# 密码学

# 1.1 密码学基本概念

密码学： 网络安全，信息安全，区块链 所必须

核心： 就是混淆真实信息，并且让合法的用户从混淆后的信息中得到真实信息

# 1.2 密码学的历史

# 1.2.1 古典密码学

① 替换法

使用固定的信息，将原文替换成密文

例如： bee，将 b 替换成 w ，e 替换 p，经过替换后变为： wpp

替换法的加密方式：

  • 单表替换： 原文和密文使用的是同一张表

  • 多表替换：表示有多张表，原文和密文进行对比

    例如： 表单 1 abcde-swtrp, 表单 2： abced-chfhk 表单 3： abcde-jftou

    原文： bee

    密钥： 312

    密文： fpk

② 移位法

将原文中所有的字母按照字母表上面的位置，进行移动

凯撒加密法：

例如： abcde 往后面移动 2 位 cdefg

# 1.2.3 现代密码学

  1. 散列函数

    • MD5

    • SHA-256

  2. 对称加密

    对称加密的加密方式和解密方式，使用的是同一把密钥

    流加密： 是对信息流中的每一个元素作为基本的处理单元进行加密

    块加密： 将原文信息分成若干组，然后按照快加密

  3. 非对称加密

    解密与加密使用的是不同的密钥

# 1.4 Byte 和 bit

Byte： 字节，数据存储的基本单位，比如硬盘 1T，单位就是字节

bit: 比特，又叫位。一个 0 或者 1 都是一个位

八个 比特 组成 一个字节
