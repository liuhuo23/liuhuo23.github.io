# Rust 简介

Rust 语言：

Rust is blazingly fast and memory-efficient: with no runtime or garbage collector, it can power performance-critical services, run on embedded devices, and easily integrate with other languages.

官方网站： https://www.rust-lang.org/

Rust 是一种系统级编程语言，注重高性能、可靠性和生产力 [1]，支持结构化编程、函数式编程、面向对象编程等多种编程范式。

Rust 是编译型语言，没有运行时 (Runtime) 和垃圾回收 (Garbage Collector)。Rust 使用所有权机制来实现自动内存管理，并以此来保证内存安全。Rust 是开源项目，目前托管在 Github 上，Rust 和所有其他官方项目都采用 Apache 许可证 2.0 和 MIT 许可证双重许可 [2]。Rust 官方软件包管理器为 Cargo。

# 安装 rust

打开网页 https://www.rust-lang.org/tools/install

下载完成后双击，回弹出一个控制台，一路默认就可以。

如果你不设置 rust 环境变量，那么 rust 会自动安装到 C/users/【用户名】下.cargo 和 .rustup

# 创建一个 rust 项目

打开控制台 -》 输入 cargo new 【项目名】 创建一个 应用程序

# match 匹配

如何来确定一个枚举子类，并根据这个子类来进行处理

如果有没有匹配到的选项，程序就会卡住，因为不知道该怎么做？有两种情况： 强制向下执行，那会遇到无法预料的问题。要么就退出程序，rust 采用了第二种。

因此使用 match 时必须要处理不知道的情况或者就必须匹配到所有的情况。

注意: match 每一种匹配必须一致，要么都不返回，要么返回相同的类型
