# 1： 计算机基本知识总结

# 1. 计算机组成

一个完整的计算机系统，是由硬件系统与软件系统两大部分组成

# 1.1： 硬件系统

计算机由运算器，存储器，控制器，输入输出设别这几个基本部分组成

  1. 中央处理器
  2. 负责数据的算数运算和逻辑运算，以及数据的加工处理，是整个计算机的核心。所有的硬件设备都是在中央处理器的控制下工作
  3. 存储器
    1. 存储数据。分为： ROM 只读存储器，RAM 随机存储器俗称内存； 外存，包括硬盘，光盘，软盘等一切能够长久存储数据并能够被计算机读取的记忆设备
  4. 输入输出设备
    1. 输入设备： 泛指一切能够将信息转换为二进制数据并被电脑处理的设备，例如：键盘，听筒，摄像头
    2. 输出设备： 泛指一切能够将二进制数据转换为非二进制信息的电子设备，例如：屏幕，喇叭

# 1.2： 软件系统

能实现特定功能的计算机指令集。包括，操作系统，数据库管理软件，驱动，应用软件

# 2： 进制转换

计算机中所使用的只有二进制，为了表示与书写记忆的方便，人们提出了八进制，16 进制

注意： 不同进制只是对同一件事物的不同描述，本质上是一样的。

# 2.1： 什么是进制

进制就是进位计数制，通俗的来说就是从右到左的数字代表的意义不同。例如： 十进制的 123，从右到左，3 代表 3 个，2 代表 20 个，1 代表一百个。分别是 100 ,101 ,102 .

那么通过十进制我们可以不妨大胆假设，对于 N 进制来说（N>=2）, 从右到左分别代表 N0,N1 ,N2 ,Nn 。所遵循的原则都是逢 N 向高位进 1.

# 2.2： N 进制转 M 进制

N 进制转 M 进制可以通过 10 进制作为中介进行转换。

10 进制转 N 进制的方法是： 用 10 进制数一直除以 N，每一次结果保留余数，所得的商继续除以 N 直到最后的商小于 N。最后的商为最左边的数，剩下的余数从下到上按照从左到右的顺序列出，最后的结果即为 N 进制数

N 进制转 10 进制数： 从右到左每一个数乘以它所在位的 Nn , 最后把结果加起来就是。

# 3: 服务，协议，端口

服务，协议，端口 。全世界的计算机正是通过他们为人类服务着

# 3.1: 常见的计算机服务

计算机的服务是一种应用程序类型，这种程序没有 UI 界面。或者这种程序可以与有 UI 界面的程序发生交互。

端口： 操作系统为了解决多个应用软件与一个 IP 之间的问题

协议： 数据传输，解析所必须遵守的数据组织格式或者处理算法。

# 4：DOS 命令

windows 常用 DOS 命

# 4.1: 常见的 DOS 命令

    • gpedit.msc // 组策略
    • lusrmgr.msc // 本地用户和组
    • services.msc // 本地服务设置
    • taskmgr // 任务管理器
    • mmc // 控制台
    • regedit // 注册表编辑器
    • msconfig // 系统配置（引导。服务。启动。工具）
    • mstsc // 远程桌面
    • mspaint // 画图
    • write // 写字板
    • notepad // 打开记事本
    • calc // 打开计算器
    • explorer // 打开文件资源管理器
    • logoff // 注销命令
    • shutdown //60 秒倒计时关机
    • ipconfig /all // 查看网络详细信息
    • set // 显示当前所有环境变量
    • dir // 列出当前目录的文件
      • dir /a // 列出所有文件包括隐藏文件
    • whoami // 查看当前计算机名及当前用户名
    • tasklist // 列出当前所有服务的详细信息
      • tasklist -svc // 列出当前所有服务对应的 PID
    • systeminfo // 查看计算机详细信息
    • tree // 以树状的形式显示文件及文件夹
    • type 1.txt // 查看文件内容
    • start // 重新打开一个 cmd 窗口
      • start 1.txt // 使用记事本打开 1.txt 文件
    • more 1.txt // 分屏查看文本内容
    • arp -a // 查看网内主机
    • net user // 查看当前计算机所有用户
      • net user as-before // 查看指定用户属性
    • net start // 查看启动的 windows 服务
      • net start 迅雷下载基础服务（用于快速申请磁盘空间及接管浏览器下载请求）
      • **net stop 迅雷下载基础服务（用于快速申请磁盘空间及接管浏览器下载请求）**PS:(需要管理员权限)
    • net view // 显示域列表、计算机列表或指定计算机的共享资源列表
      • net view ip // 显示指定的 ip 的共享资源列表
    • net config // 用于显示服务器的打开文件数量和空闲回话时间及用户登录数量
      • net config Server // 查看服务器的 PS:(运行需要管理员权限)
      • net config Workstation // 查看当前工作站的 PS:(运行需要管理员权限)
    • net share // 查看本地开启的共享文件
    • netstat -an // 查看开启的端口信息
      • netstat -ano // 查看带有 PID 的端口信息
    • nslookup // 查询 DNS 的记录，查询域名解析，用来诊断网络问题
    • tracert www.baidu.com // 路由追踪
    • ping // 测试网络连通
    • cd // 切换目录
      • cd . // 还在当前目录
      • cd .. // 切换到上层目录
      • *cd * // 切换至根目录
      • cd . > 1.txt // 创建文件
    • d： // 直接跳转至 d 盘根目录
    • md test // 创建 test 目录
    • rd test // 删除 test 目录
      • rd /s test // 删除目录需要确定
    • copy 1.txt 2.txt // 复制 1.txt 的内容到 2.txt，如果 2.txt 存在内容会提示你是否覆盖
    • del 2.txt // 删除文件
      • del -f 1.txt // 强制删除
    • move 1.txt test // 将 1.txt 移动到 test 文件夹内
    • move 1.txt 2.txt // 将 1.txt 重命名为 2.txt
    • ren 1.txt 2.txt // 重命名
    • shutdown -s // 一分钟后关机
      • shutdown -s -t 100 -c "bye" // 100 秒后关机并弹出 bye 消息
      • shutdown -a // 取消计划的关机任务
    • net use K: \ip\c$ // 映射目标 ip 的 c 盘到本地 k 盘
    • net use k: \ip\c$ \del // 删除映射的 k 盘
    • taskkill /f /im xx.exe // 强制终止某进程
    • findstr “Hello”aa.txt // 在 aa.txt 文件中寻找字符串 hello
    • echo hello » pass.txt // 将 hello 追加到 pass.txt 文件中 (一个> 直接覆盖原内容)
    • color 02 // 把 cmd 中的字改为绿色字体
