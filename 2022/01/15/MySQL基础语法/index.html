# 1. 数据库操作

基本的数据库操作，包括数据库连接，数据库基本语法

SQL 是用于访问和处理数据库标准的计算机语言

SQL 指结构化查询语言，全称是 Structured Query Language

# 2. 数据库常用函数

# 2.1 updatexml() 语法 updatexml (目标 xml 文档，xml 路径，更新的内容)

语法： select updatexml('anything',concat('~',(select version())),'xxx');

# 2.2 substring() 函数， left() , mid() 函数

语句： select substring(user(),1,2);

# 2.3 if(a,b,c) 如果 a 为真，则返回 b，否则返回 c

语句： select if(length(database())=5,1,0);

# 2.4 hex() && unhex() 函数

渗透中有些注入点因为数据库中对数据字段的类型定义，可能不支持 union 来显示某些不同类型的内容，所以使用 hex 对数据进行 16 进制的编码

hex() 参数可以是任何类型
