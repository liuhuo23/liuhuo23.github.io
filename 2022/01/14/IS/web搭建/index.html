# 一：Web 基础知识

# 1. 网站的分类

  1. asp： 多用于小型站点，企业网站，个人博客

  2. jsp： 大型商业网站

  3. php： 多数用于中小型网站

  4. aspx： 多数用于小型站点，地方政府

# 2. 域名简介

域名：（英语： Domain Name）

每一个域名都对应一个唯一的公网 IP

# 二：WEB 容器

# 1. web 容器

web 容器是中间件，负责处理请求

常见的 web 容器： IIS,Apache,Nginx,Tomcat,Jboss

# 2. 脚本语言

脚本语言是一种 web 端的网页源代码，它又分为动态网页和静态网页

动态网页： jsp，php

静态网页： html,heml5,shtml

# 3. 数据库

  1. MySQL：开源的免费的轻量的数据库，一般被使用在大中小型网站

  2. Oracle： 付费的，一般用于大型网站

  3. Micsoft SQLServer： 适用中小型的网站

# 三： 服务器

# 1. 简介

服务器是一种高性能计算机，作为网络节点，多提供文件存储服务

# 2. 常见的服务器搭配

windows + IIS + Access + asp/aspx

windows + IIS + SQLServer + asp/aspx

windows + Apache + MySql + PHP

windows + Nginx + MySql + JSP

linux + Tomcat + jsp + MySql

linux + Apache + PHP + MySql (LAMP)

linxu + Nginx + MySql + PHP (LNMP)

linux + Nginx + Oracle + JSP

# 3. 服务器搭建

Windows：

Windows server2000 IIS6.0

Windows server2003 IIS7.0

Windows server2008 IIS7.0/7.5

Windows server2012 IIS8.0

Windows server2016 IIS8.5

Windows10 IIS10.0

Linux：

Ubuntu

Red Hat Enterprise Linux (RHEL)

SUSE Linux Enterprise Server

CentOS (Community OS) Linux Server

Debian

# 4. 数据库、脚本与系统的关系

# 4.1 标配

Access + asp/aspx + IIS+Windows

Sqlserver(mssql) + asp/aspx + IIS + Windows

Mysql + php/jsp + Apache + Linux

Oracle +jsp + apache/Nginx + Linux

​ PS

​ 数据库与脚本、系统 、并非绑定关系，只是出于性能而已的标配、另外还要注意 Windows 对大小写不敏感，而 Linux 对大小写敏感。
